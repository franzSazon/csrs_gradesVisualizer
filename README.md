# UPMin CSRS Grades Visualizer and auto GWA calculator 
UPMin specific tool that mitigates the time-consuming labor of manually inputting our grades into a GWA calculator.

## How to use this thingy?

Hit Ctrl + s on the "Grades" Section of the UPMin CSRS website and save the HTML file. Upload said HTML on the Streamlit web app to automatically calculate your GWA as well as include nifty visualizations of your academic performance.
